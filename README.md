# Pterosaur

![Pterosaur Logo](logo.png)

Pterosaur is a command-line tool designed for email identification and verification. It allows users to scrape email addresses from specified domains, verify individual email addresses, and search for emails associated with a domain using the Hunter.io API.

## Features

- **Identify Emails**: Scrape email addresses from a specified domain.
- **Verify Email**: Check the validity of an email address using the Hunter.io API.
- **Domain Search**: Find emails associated with a specific domain using the Hunter.io API.

## Requirements

- Python 3.6 or higher
- `httpx`
- `beautifulsoup4`
- `python-dotenv`
